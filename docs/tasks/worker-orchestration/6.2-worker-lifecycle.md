## Done When

- Workers spawn automatically for unassigned cards using the resolved provider
- Workers advance through pipeline steps via complete_step
- Workers can finish, wont-do, or ask-user
- Crashes trigger recovery on the same session
- Continue-retry nudges workers that don't signal intent
- All worker hooks fire
